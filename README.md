# 🎓 Student Profile Website

This project is a simple and responsive **Student Profile Website** created using **HTML and CSS**.  
It showcases basic student details such as profile picture, personal information, and skills/hobbies.

---

## 📌 Project Overview

The Student Profile Website is designed as a beginner-friendly frontend project to practice:
- HTML page structure
- CSS styling
- Layout techniques using Flexbox
- Box model concepts

This project is suitable for learning and demonstrating basic frontend development skills.

---

## 🚀 Features

- 👤 Profile picture section  
- 🧑‍🎓 Student name, roll number, and class details  
- ⭐ Skills / hobbies listed clearly  
- 🎨 Styled using CSS colors, fonts, and box model  
- 📱 Simple responsive layout using Flexbox  

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
  - Box Model
  - Flexbox
  - Basic Responsive Design

---

## 📚 What I Learned

- Creating structured webpages using HTML  
- Styling elements with CSS  
- Using Flexbox for layout alignment  
- Applying box model properties (margin, padding, border)  
- Organizing frontend project files
